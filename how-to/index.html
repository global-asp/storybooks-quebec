Lire des contes

  • Cliquer sur un conte pour le lire.
  • Cliquer sur Changer de langue pour lire le conte dans une autre langue.
  • Cliquer sur le icône fr pour passer rapidement de n’importe quelle langue à français.
    • Veuillez noter que cela change seulement la langue du texte. Cliquer sur Changer de langue pour changer aussi la langue de la version audio.

Écouter des contes

  • Cliquer sur l’icône à côté de chaque texte sur la page pour écouter le texte lu à voix haute.
    • Pour faire une pause ou recommencer l’audio à n’importe quel moment, cliquer sur l’icône .
  • Pour écouter le conte en entier lu à haute voix, cliquer sur les commandes audio en haut de la page.
    • Si vous ne voyez aucune commande audio, ou si la commande audio est grisée, cela signifie qu’il n’y a pas de version audio disponible dans cette langue actuellement.
  • Vitesse audio : Utiliser le curseur sous les commandes de lectures pour ajuster la vitesse de lecture. Cliquer sur l’icône pour retourner à la vitesse normale (1×).

Télécharger des contes

  • Pour télécharger chaque conte, cliquer sur le bouton Téléchargement PDF en haut ou en bas de la page.
  • Pour télécharger l’audio complète pour chaques contes, Clique-droit sur les commandes audio en haut de la page et sélectionner Enregistrer le fichier audio sous… pour enregistrer le fichier audio dans votre ordinateur.
    • En fonction de votre système d’exploitation et de votre navigateur, il y a peut-être, sur votre panneau de commandes audio, une icône pour télécharger sur laquelle vous pouvez cliquer pour enregistrer le conte sur votre appareil.
